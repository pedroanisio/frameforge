# docker/fonts.apt.txt — Debian font packages installed into the FrameForge image.
#
# This is the script/CJK/emoji/symbol coverage that the google/fonts corpus
# (installed separately, see docker/collect-google-fonts.sh) does NOT ship well:
# full Noto, the big CJK families, Indic/Arabic/Thai/Sinhala/Khmer scripts, and
# emoji/symbol fallbacks. google/fonts covers the Latin display/text/mono breadth.
#
# One package per line. Comments (`#`) and blank lines are ignored by the build.
# The Dockerfile installs these in bulk, then retries individually so a name that
# is absent on a given Debian release is skipped instead of aborting the build.
# For the absolute maximum, build with --build-arg FONTS_APT_WILDCARD=1 (adds
# every `fonts-*` package apt knows about).
